# Daily AI News Curation

AI 관련 뉴스, 논문, 오픈소스 프로젝트, 인사이트를 매일 자동으로 큐레이션합니다.

## 웹으로 보기

- [https://MoriochoRadio.github.io/ai-news-curation/](https://MoriochoRadio.github.io/ai-news-curation/)

## 리포 사용 방법

- 이 저장소를 `Watch` 해두면 매일 자동으로 업데이트를 받을 수 있습니다.
- 로컬에 복사하려면 아래 명령을 사용하세요.

    git clone https://github.com/MoriochoRadio/ai-news-curation.git

## 자동화

- 매일 아침 9시(UTC) GitHub Actions가 실행되어 새로운 큐레이션이 생성됩니다.
- 큐레이션 결과는 `daily/YYYY-MM-DD.md` 파일로 저장됩니다.

## 기여 / 피드백

관심 있는 주제, 새로운 소스, 개선 의견은 이슈나 PR로 자유롭게 남겨주세요.
